## Responsive Behavior

### Mobile (< 640px)
- Legend: Smaller size
- Text: Reduced to text-xs
- Buttons: Full width where applicable
- Markers: 8x8 (w-8 h-8)

### Tablet (640px - 1024px)
- Legend: Medium size
- Text: text-sm
- Markers: 10x10 (w-10 h-10)

### Desktop (> 1024px)
- Legend: Full size
- Text: text-sm/base
- Markers: 10x10 (w-10 h-10)

## Animation Specifications

### Pulse (Nearby Hazards)
```css
animation: pulse
duration: 2s
easing: cubic-bezier(0.4, 0, 0.6, 1)
iteration: infinite
```

### Ping (Route Hazards)
```css
animation: ping
duration: 1s
easing: cubic-bezier(0, 0, 0.2, 1)
iteration: infinite
```

### Slide In (Proximity Alert)
```css
animation: slide-in-from-bottom-4
duration: 0.3s
easing: ease-out
```
