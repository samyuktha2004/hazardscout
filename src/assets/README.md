# Assets Directory

This folder contains static assets for the Hazard Scout application:

## Images
- Vehicle images (VW Virtus, etc.)
- Hazard icons
- Brand logos
- UI graphics

## Usage
Import images using:
```typescript
import vwVirtusImage from '../assets/vw-virtus.jpg';
```

## Recommended Structure:
```
assets/
├── images/
│   ├── vehicles/
│   │   └── vw-virtus.jpg
│   ├── hazards/
│   │   ├── pothole-icon.svg
│   │   ├── construction-icon.svg
│   │   └── accident-icon.svg
│   └── brand/
│       └── vw-logo.svg
├── icons/
└── sounds/
    └── notification-beep.mp3
```
