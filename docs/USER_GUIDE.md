# Hazard Scout User Guide

## Core Features

### 1. Hazard Detection & Reporting

- View hazards on the map
- Report new hazards
- Confirm existing hazards
- Mark hazards as resolved

### 2. Navigation

- Plan routes with hazard awareness
- Real-time hazard alerts
- Live navigation with safety updates

### 3. Safety Features

- Severity-based hazard marking
- Community verification system
- Auto-resolution of old hazards
- Push notifications for nearby hazards
